# Nishikant | Portfolio

Personal portfolio website built using HTML, CSS, and JavaScript.
Designed to be responsive, simple, and functional across devices.

## Project Overview

This portfolio showcases my projects, skills, and frontend fundamentals.
Built without frameworks, focusing on core web technologies and clean structure.

## Features

- Fully responsive layout (mobile and desktop)
- Dark / Light theme toggle with saved preference
- Working mobile navigation menu
- Smooth scrolling between sections
- Contact form (UI only)
- Minimal and clean design

## Tech Stack

- HTML5 – Structure
- CSS3 – Styling and layout
- JavaScript – Theme toggle and menu logic
- Ionicons – Icons
- Google Fonts – Inter, Montserrat
